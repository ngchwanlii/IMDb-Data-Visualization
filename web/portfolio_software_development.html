EatAllDay link

EatAllDay is a food delivery application. We've put great local food all in one place and you can order food from your favorite restaurants and have it delivered to your doorstep! Fast delivery with real-time GPS tracking!

Github Link link

Status: On Processing

Project Details:
This is my personal project developed in summer of 2017. My initial intention is to develop a food delivery application for my lovely hometown - Kota Kinabalu, Sabah. We'll be planning to expand and reach out more restaurants and cities in the future!

Learning Outcome:

  • learn how to use Django framework
  • learn how to host application in Heroku
  • learn how to use SQLite
  • learn how to test RESTful API using Postman
  • learn how to develop an iOS application written in Swift

    • Technologies Stack: Django, Python, SQLite, Swift
